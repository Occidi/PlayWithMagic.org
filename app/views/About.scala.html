We're Looking for a Few Good Routines

About this Website

Play With Magic is a place for Magicians to catalog and share their Routines, find new Routines, and build Sets.

Once an account is created, a Magician can post their own Routines, or Magic Tricks, search posted Routines, and keep track of other Routines that they like.

From this list, a Magician can then create a Set, or a series of Routines. The Set Builder allows a Magician to receive suggestions based on Venue, Audience Size, Desired Effects, and so forth.

Responsive Layout built on Twitter Bootstrap

Built with Java on the Play Framework

Backend Database by PostgreSQL

Hosted by Heroku
